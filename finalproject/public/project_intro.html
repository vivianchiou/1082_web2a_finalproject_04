作業格式：
1. 請自行找一個 bootTstrap template，版本要 4.* 以上，不能跟老師上課 demo 的相同。以下是可以參考的網站：
2. 請在該 bootstrap template 之 選單最上層，加入 TKU 下拉式選單，請將課堂實作4個 demo 放 入TKU 選單中。請修改css格式，讓每個都螢幕寬度改變時，都可以對中，內容跟上選單要保持適當距離。
3. 請在TKU選單中，加入一個 Github項目，要顯示兩張圖片，第一張圖片為git bash 成功送上 Github的圖片，第二張圖片為Github repository 成功上傳，可以顯示在根目錄下檔案目錄的圖片。
4. 請在 TKU 選單中，加入Ｍy Class (我的課表)及 Ｍy Resume(我的履歷表)，格式自定，以資訊完整且美觀為原則。
5. 在 TKU選擔下，產生一個選項 Project說明，清楚說明本作業你所完成的部分，尤其是上課沒有教，你自主學習完成的部分。
6. 以上從 TKU 選單內之項目，共有8項，每項最上面都必須要有 bootstrap 的選單。畫面格式要適當修改，內容可以對中，可跟隨螢幕寬度而變動。 作業心得：
做作業的途中，遇到蠻多問題，例如網頁滑不動，格式跑掉等等，需要很有耐心的一個一個上網查詢並解決問題。
過程中，更懂得表格格式的應用，以及表格的處理多樣性，也另外去學習去呈現我所想的，發現其實關鍵字都差不多，有時間的話可以邊做邊試試看，有時會有額外的收穫，也因為永遠學不完，因此覺得格外有趣，會再努力學習增進自己！
